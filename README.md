## 说明

- 这是一个正则表达式代码片段插件
- 这个插件的正则表达式足以满足您工作大部分需求
- 只要您有一点正则基础都能看得懂，并且能够从中学习。
- 支持的页面：javascript、vue、html



## 使用教程

- 连续按下两次 **`S`** 键 呼出校验数字的正则表达式
- 连续按下两次 **`Z`** 键 呼出校验字符的正则表达式
- 连续按下两次 **`T`** 键 呼出特殊需求的正则表达式



## 校验数字的正则表达式

- m到n位的数字

  ```
  /^\d{m,n}$/
  ```

- n位的数字

  ```
  /^\d{n}$/
  ```

- 带1到2位小数的正数或负数

  ```
  /^(\-)?\d+(\.\d{1,2})?$/
  ```

- 数字

  ```
  /^\d+$/
  ```

- 有1~3位小数的正实数

  ```
  /^[0-9]+(.[0-9]{1,3})?$/
  ```

- 有两位小数的正实数

  ```
  /^[0-9]+(.[0-9]{2})?$/
  ```

- 正数、负数、和小数

  ```
  /^(\-|\+)?\d+(\.\d+)?$/
  ```

- 正浮点数

  ```
  /^[1-9]\d*\.\d*|0\.\d*[1-9]\d*$/
  ```

- 浮点数

  ```
  /^(-?\d+)(\.\d+)?$/
  ```

- 至少n位的数字

  ```
  /^\d{n,}$/
  ```

- 负浮点数

  ```
  /^-([1-9]\d*\.\d*|0\.\d*[1-9]\d*)$/
  ```

- 零和非零开头的数字

  ```
  /^(0|[1-9][0-9]*)$/
  ```

- 非正整数

  ```
  /^-[1-9]\d*|0$/
  ```

- 非正浮点数

  ```
  /^((-\d+(\.\d+)?)|(0+(\.0+)?))$/
  ```

- 非负整数

  ```
  /^\d+$/
  ```

- 非负浮点数

  ```
  /^\d+(\.\d+)?$/
  ```

- 非零开头的最多带两位小数的数字

  ```
  /^([1-9][0-9]*)+(.[0-9]{1,2})?$/
  ```

- 非零的正整数

  ```
  /^[1-9]\d*$/
  ```

- 非零的负整数

  ```
  /^-[1-9]\d*$/
  ```



## 校验字符的正则表达式

- 汉字

  ```
  /^(?:[\u3400-\u4DB5\u4E00-\u9FEA\uFA0E\uFA0F\uFA11\uFA13\uFA14\uFA1F\uFA21\uFA23\uFA24\uFA27-\uFA29]|[\uD840-\uD868\uD86A-\uD86C\uD86F-\uD872\uD874-\uD879][\uDC00-\uDFFF]|\uD869[\uDC00-\uDED6\uDF00-\uDFFF]|\uD86D[\uDC00-\uDF34\uDF40-\uDFFF]|\uD86E[\uDC00-\uDC1D\uDC20-\uDFFF]|\uD873[\uDC00-\uDEA1\uDEB0-\uDFFF]|\uD87A[\uDC00-\uDFE0])+$/
  ```

- 由英文、数字、汉字组成且字符长度在m到n之间

  ```
  /^[\u4E00-\u9FA5A-Za-z0-9]{m,n}$/
  ```

- 由英文开头且由英文、数字、下划线组成且字符长度在m到n之间

  ```
  /^[A-Za-z][A-Za-z0-9_]{m,n}$/
  ```

- 由英文、数字、特殊字符组成且至少包括1个大写字母、1个小写字母、1个数字、1个特殊字符且字符长度在m到n之间

  ```
  /^\S*(?=\S{m,n})(?=\S*\d)(?=\S*[A-Z])(?=\S*[a-z])(?=\S*[!@#$%^&*? ])\S*$/
  ```

- 英文字母 (大写)

  ```
  /^[A-Z]+$/
  ```

- 英文字母 (小写)

  ```
  /^[a-z]+$/
  ```

- 英文字母

  ```
  /^[A-Za-z]+$/
  ```

- 数字和字母组成

  ```
  /^[A-Za-z0-9]+$/
  ```

- 长度在m到n位的字符

  ```
  /^.{3,20}$/
  ```

- 由数字、英文、下划线组成

  ```
  /^\w+$/
  ```

- 由中文、英文、数字、下划线组成

  ```
  /^[\u4E00-\u9FA5A-Za-z0-9_]+$/
  ```
