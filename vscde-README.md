## 说明

- 一个正则表达式代码片段插件

- 正则表达式包括：数字类正则、字符类正则、特殊类正则

- 支持的页面：javascript、vue、html
- 所有的正则表达式已放在Github仓库：https://github.com/learnguos/vscode-regex



## 使用教程

- 按两次 **`S`** 键 呼出校验数字的正则表达式

- 按两次 **`Z`** 键 呼出校验字符的正则表达式

- 按两次 **`T`** 键 呼出特殊需求的正则表达式
